# Курс: Распознавание и синтез речи

Краткие выжимки из лекций по обработке речи, распознаванию речи (ASR) и синтезу речи (TTS).

## Лекции

- [Лекция 1: Основы обработки речи](./lec1.md)
  - Основы звука и речи
  - Речевой и слуховой аппарат человека
  - АЦП и предобработка

- [Лекция 2: Подготовка данных для обучения ASR](./lec2.md)
  - Манифесты данных для NeMo
  - Токенизация (BPE, Unigram)
  - Модели ASR и их сравнение

- [Лекция 3: Адаптеры для дообучения](./lec3-adapter.md)
  - Архитектура адаптеров
  - Ограничения и преимущества
  - Риски переобучения

- [Лекция 4: Синтез речи (TTS)](./lec4-prez4_sitez.md)
  - Исторический контекст (Кемпелен, Гельмгольц, Фант)
  - Модель "Источник — Фильтр"
  - Вокодер VODER
  - Статистический и конкатенативный синтез

- [Лекция 5: Продвинутый синтез речи - Часть 2](./lec5-tts_lecture2-2.md)
  - Метрики качества (MOS, WER, Speaker similarity)
  - LM-based модели
  - VALL-E, GSLM, AudioLM
  - Дискретное моделирование в аудио
